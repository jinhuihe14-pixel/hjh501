# 汽车美容门店管理系统

全栈汽车美容门店管理系统，支持前台开单、技师移动端、店长管理后台、车主微信小程序四端联动。

## 功能特性

### 🛠️ 店长管理后台
- **数据概览**：实时查看门店营收、订单统计
- **订单管理**：查看所有订单，跟踪订单状态
- **员工管理**：管理员工账号和权限
- **服务项目**：配置洗车、精洗、镀膜等服务项目
- **商品管理**：管理车载香薰、坐垫等汽车用品
- **薪资规则配置**：自定义各岗位薪资参数（底薪、提成、奖金、扣款）
- **薪资核算**：自动计算员工工资，支持手动调整
- **报表统计**：营收报表、客户分析、员工业绩
- **保养提醒**：管理车辆保养提醒

### 💼 前台开单端
- **开单收银**：快速创建服务订单，支持选择服务项目和商品
- **订单列表**：查看订单记录
- **客户管理**：管理客户信息和车辆档案
- **保养提醒**：查看待提醒的保养车辆

### 📱 技师移动端
- **我的工单**：查看待处理工单，更新施工状态
- **业绩查询**：查看工时统计和产值
- **薪资查看**：查看本月工资明细

### 🛒 销售导购端
- **业绩查询**：查看销售业绩和提成
- **薪资查看**：查看工资明细

## 技术栈

- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **数据库**: PostgreSQL
- **ORM**: Prisma
- **样式**: Tailwind CSS
- **认证**: JWT + HTTP-only Cookie
- **表单**: React Hook Form + Zod

## 快速开始

### 环境要求
- Node.js 18+
- PostgreSQL 14+

### 安装依赖

```bash
npm install
```

### 配置数据库

修改 `.env` 文件中的数据库连接信息：

```env
DATABASE_URL="postgresql://postgres:password@localhost:5432/car_beauty_shop?schema=public"
```

### 初始化数据库

```bash
# 生成 Prisma Client
npm run db:generate

# 推送数据库 schema
npm run db:push

# 填充初始数据
npm run db:seed
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

## 核心业务流程

### 1. 薪资核算流程
1. 店长配置各岗位薪资规则（次月生效）
2. 每笔订单自动拆分业绩数据到对应员工
3. 月末点击"计算薪资"自动生成工资单
4. 店长可手动调整奖惩金额
5. 确认锁定后数据不可修改

### 2. 订单流程
1. 前台开单：选择客户、车辆、服务项目、商品
2. 指定施工技师和销售导购
3. 技师接收工单，开始施工
4. 施工完成，更新订单状态
5. 业绩自动计入对应员工台账

### 3. 保养提醒流程
1. 录入车辆保养记录（里程、保养项目）
2. 系统自动计算下次保养时间/里程
3. 临近保养周期自动出现在提醒列表
4. 前台联系客户后标记已提醒

## 薪资计算规则

| 岗位 | 薪资构成 |
|------|----------|
| 前台收银 | 底薪 + 开单总额 × 提成比例 + 全勤奖 - 迟到扣款 |
| 美容技师 | 底薪 + 工时 × 工时单价 + 全勤奖 - 迟到扣款 |
| 销售导购 | 底薪 + 商品销售额 × 提成比例 + 全勤奖 - 迟到扣款 |
